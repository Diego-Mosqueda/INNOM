# INNOMMX Landing Page

## Descripción del Proyecto

Landing page moderna y minimalista para INNOMMX, empresa líder en soluciones de pulverización industrial. La página está diseñada para mostrar los productos y servicios de la empresa a las industrias alimenticia, cosmética y farmacéutica.

## Características Principales

- **Diseño Minimalista**: Interfaz limpia con espacios en blanco generosos y jerarquía visual clara
- **Glassmorphism**: Efectos de vidrio translúcido en el navbar con blur al hacer scroll
- **Animaciones AOS**: Animaciones sutiles de entrada en todas las secciones
- **Totalmente Responsivo**: Optimizado para dispositivos móviles, tablets y desktop
- **Scroll Spy**: Navegación activa que detecta la sección visible
- **WhatsApp Flotante**: Botón de contacto directo siempre visible

## Tecnologías Utilizadas

- **HTML5**: Estructura semántica
- **CSS3**: Estilos personalizados con efectos glassmorphism
- **Bootstrap 5.3**: Framework CSS para grid y componentes
- **React 18**: Componentes interactivos
- **AOS**: Animaciones on scroll
- **Bootstrap Icons**: Iconografía minimalista
- **Google Fonts (Inter)**: Tipografía moderna y legible

## Secciones de la Página

1. **Hero**: Carrusel de imágenes a pantalla completa con call-to-action
2. **Acerca**: Información sobre la empresa, historia e industrias servidas
3. **Maquinaria**: Catálogo de equipos (Agitadores, Mezcladoras, Molinos, Tanques, Tolvas)
4. **Servicios**: Servicios especializados ofrecidos
5. **Contacto**: Formulario, información de contacto, redes sociales y mapa
